# SimpleCompass

A Kenshi mod that adds a configurable compass HUD element to show your current heading in the game world.

Get it from your preferred modding platform:
- **Steam Workshop**: [TBD:SimpleCompass](https://steamcommunity.com/sharedfiles/filedetails/?id=TBD)
- **NexusMods**: [TBD:SimpleCompass](https://www.nexusmods.com/kenshi/mods/TBD)

## Features

- **Three compass modes** - Click the compass button to cycle between display modes:
  - **Full** - A rotating compass strip with cardinal/intercardinal tick labels, a red caret indicator, and the exact bearing in degrees with direction name.
  - **Number with Direction** - Compact display showing the bearing in degrees with direction name (e.g. "45(NE)").
  - **Direction Only** - Minimal display showing only the cardinal or intercardinal direction (e.g. "N", "SE").
- **Always available** - The compass is always visible (no selection required) and appears centered near the top of the screen.
- **Persistent mode** - Your chosen display mode persists for the game session (must be re-selected on restart).
